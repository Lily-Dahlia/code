#  Simple URL Shortener

Este es un script de automatización ultra ligero escrito en **Python** que te permite acortar enlaces largos utilizando la API de TinyURL sin necesidad de instalar librerías externas o dependencias pesadas (`requests`, etc.).

Ideal para limpiar enlaces eternos antes de compartirlos en tus redes sociales o proyectos.

##  Características

* **Sin dependencias externas:** Utiliza únicamente módulos nativos de Python (`urllib`).
* **Doble modo de uso:** Funciona de manera interactiva o mediante argumentos de consola.
* **Rápido y ligero:** Respuesta prácticamente instantánea.

## Requisitos

* **Python 3.x** instalado en tu sistema.
